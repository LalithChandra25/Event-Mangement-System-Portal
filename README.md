# Event-Mangement-System-Portal

The Community Event Management System Portal is a comprehensive web-based application designed to streamline and digitize the process of organizing, managing, and participating in community events. Developed using Python Flask, SQLite, HTML, CSS, and Bootstrap, the platform provides a centralized solution for event administrators and participants, eliminating the inefficiencies of traditional manual event management.

The system enables users to create accounts, securely log in, and access various event-related functionalities through an intuitive and responsive user interface. Event organizers can create, update, and manage events, define participant limits, and monitor registrations in real time. The platform incorporates a slot-based registration mechanism to ensure controlled participation and prevent overbooking.

A key feature of the system is its integrated payment management workflow. Organizers can upload QR codes for event fee collection, while participants can submit payment screenshots as proof of payment. Administrators can then verify payments directly through the portal, reducing manual verification efforts and improving transparency.

The attendance management module allows organizers to track participant registrations and attendance records efficiently. Additionally, the gallery management feature enables users to upload and view event photographs, creating a digital archive of community activities and enhancing participant engagement.

To maintain database efficiency and system performance, the application includes automated cleanup mechanisms that remove outdated event and gallery records. The responsive design, built with Bootstrap, ensures seamless accessibility across desktops, tablets, and mobile devices.

## Key Features
1. User Authentication and Session Management
2. Event Creation and Management
3. Slot-Based Participant Registration
4. Attendance Tracking and Verification
5. QR Code Payment Integration
6. Payment Screenshot Upload and Validation
7. Event Gallery Upload and Display
8. Automated Cleanup of Old Records
9. Responsive and User-Friendly Interface
## Technologies Used
**Frontend**: HTML, CSS, Bootstrap

**Backend**: Python Flask

**Database:** SQLite

**File Management:** Image and Screenshot Upload Handling
